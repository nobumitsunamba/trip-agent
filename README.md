# 出張検索エージェント

出発地・目的地・希望着時刻・出張日・ホテル予算を入力すると、  
**新幹線候補3本** と **ホテル候補3件** を自動検索し、予約ページをブラウザで開くGUIツールです。

---

## ファイル構成

```
trip-agent/
├── main.py            # 起動ファイル（tkinter GUI）
├── search_train.py    # 新幹線検索（Yahoo!乗換案内スクレイピング）
├── search_hotel.py    # ホテル検索（楽天トラベルAPI）
├── last_input.json    # 前回入力の自動保存
├── .env               # APIキー設定ファイル
├── requirements.txt   # 依存ライブラリ
└── README.md
```

---

## セットアップ

### 1. 依存ライブラリのインストール

```bash
pip install -r requirements.txt
```

### 2. 楽天トラベル APIキーの設定

[楽天ウェブサービス](https://webservice.rakuten.co.jp/) でアプリIDを取得し、  
`.env` ファイルに設定します：

```
RAKUTEN_APP_ID=your_application_id_here
```

### 3. 起動

```bash
python main.py
```

## 使い方

| 入力項目 | 説明 | 例 |
|---|---|---|
| 出発地（最寄り駅） | 新幹線の乗車駅 | `東京` |
| 目的地（最寄り駅） | 降車・目的地駅 | `名古屋` |
| 希望着時刻 | HH:MM 形式 | `10:00` |
| 出張日 | YYYY-MM-DD 形式 | `2026-06-15` |
| ホテル予算上限 | 1泊あたり（円） | `15000` |

1. 各項目を入力して **「🔍 検索する」** ボタンを押す
2. 新幹線候補3本・ホテル候補3件が並んで表示される
3. 各カードの **「予約ページを開く」** ボタンで予約サイトへ

## 機能詳細

### 新幹線検索

- **Yahoo!乗換案内** をスクレイピングして、着時刻指定で経路を検索
- 在来線区間を含むトータル所要時間・料金を取得
- 候補3本を表示（取得できない場合はYahoo!乗換案内のリンクを表示）
- **予約サイトの自動判定**：
  - 東海道・山陽新幹線方面 → **JR東海 EX予約**（https://expy.jp/）
  - 東北・北海道・上越・北陸新幹線方面 → **JR東日本 えきねっと**（https://www.eki-net.com/）

> **Note:** Yahoo!乗換案内のスクレイピングは利用規約上グレーゾーンです。  
> ブロックされた場合は「Yahoo!乗換案内で全経路を確認する」ボタンからブラウザで直接確認できます。

### ホテル検索

- **楽天トラベルAPI** (VacantHotelSearch) を使用して空室を検索
- 予算内・駅近優先（料金昇順）でフィルタリング
- 空室検索が失敗した場合は KeywordHotelSearch へ自動フォールバック
- 候補3件を表示（ホテル名・アクセス・料金・レビュー）
- 「楽天トラベルで予約する」ボタンで該当ページを開く

### 前回入力の保存

- 検索実行時に `last_input.json` へ自動保存
- 次回起動時に前回の入力値を初期値として表示

## よくある質問

**Q: ホテルが表示されない**  
A: `.env` に正しい `RAKUTEN_APP_ID` が設定されているか確認してください。

**Q: 新幹線の時刻が「—」と表示される**  
A: Yahoo!乗換案内のスクレイピングがブロックされている可能性があります。  
「Yahoo!乗換案内で全経路を確認する」ボタンから直接ブラウザで確認できます。

**Q: tkinter が見つからないというエラーが出る**  
A: Linux の場合は `sudo apt install python3-tk` でインストールしてください。
